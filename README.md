# ⚽ Arena Futebol Store

Mini-site institucional desenvolvido como exercício prático de HTML e CSS, com o uso de IA.

O projeto simula o site de uma empresa de artigos de futebol, com layout moderno, responsivo e estrutura organizada.

---

## 📌 Sobre o Projeto

A **Arena Futebol Store** é um site institucional fictício criado para representar uma loja especializada em artigos de futebol.

O objetivo foi desenvolver:

- Estrutura em HTML
- Estilização externa com CSS
- Layout centralizado
- Menu com efeito hover
- Cards com borda arredondada e sombra
- Botões com efeito hover
- Site responsivo (cards empilhados em telas menores)

## 📱 Responsividade

O site foi desenvolvido com foco em responsividade:

- Layout centralizado com `max-width`
- Grid adaptável
- Cards empilhados em telas menores
- Menu ajustável para diferentes tamanhos de tela
